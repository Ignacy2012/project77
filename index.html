how to set up visual code studio:

step 1.Open Visual Studio IDE software.

step 2.Create a new folder on your computer.

step 3.Open Visual Studio IDE and click on File > Open Folder, then browse and open the folder created in step 2.

step 4.Install Live-Server Extension if not already installed.


The console screen is part of the element section and is connected with javascript.

You can run JavaScript in the Console to interact with the page that you're are inspecting.


how to open the console screen:

step 1. right click on the page *not on the text*

step 2. click the word "inspect"

step 3. in the element pannel look at the top and click "console"


How to print anything on the console pannel:

step 1. go to javascript

step 2. enter the string of code for a name "var *variable name here* = "insert name here" "

step 3. press enter then enter this string of code "console.log(*variable name*)"
